The dataset used in this project was sourced from Kaggle.

* Source Link: [Chocolate Sales Dataset (2023-2024)](https://www.kaggle.com/datasets/ssssws/chocolate-sales-dataset-2023-2024/data)

* Format: CSV (Imported into MySQL for relational modeling)

⚠️ Disclaimer:
This dataset is synthetic and was developed specifically for data analysis purposes. It is designed to simulate real-world retail sales patterns, customer demographics, and product preferences for a global chocolate retailer.
